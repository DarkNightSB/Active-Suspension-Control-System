# Active-Suspension-Control-System
Comparative Study of Active Suspension Systems using P, PI, and PID Control. Simulated and validated through MATLAB/Simulink and CarSim co-simulation.

## 📊 Methodology
Dự án thực hiện so sánh 4 trường hợp:
1. **No PID (Passive):** Hệ thống treo bị động tiêu chuẩn.
2. **P Control:** Điều khiển tỉ lệ.
3. **PI Control:** Điều khiển tỉ lệ - tích phân.
4. **PID Control:** Điều khiển tối ưu nhất cho đáp ứng hệ thống.
